💡 什么是 Simple Screen Editor？

Simple Screen Editor 是一款面向 80x25 固定文本单元环境的 GUI 屏幕编辑器， 可以稳定处理韩文/英文混合输入时的字符宽度对齐。

编辑核心使用 Go 实现，桌面外壳使用 Wails。 它专注于构建可复现的文本单元编辑基础，而不是一次性的实验工具。

✨ 主要功能

📐 80x25 固定网格

屏幕固定为 80 列 x 25 行，保证一致的单元级编辑体验。

🇰🇷🇺🇸 韩文/英文宽度对齐

基于字符宽度感知，插入/删除时依旧保持光标位置与单元布局稳定。

📋 剪贴板集成

支持系统剪贴板复制/粘贴，可直接用于真实编辑流程。

💾 UTF-8 文件编辑

支持 UTF-8 文本文件打开/保存，并可进行跨平台测试。
