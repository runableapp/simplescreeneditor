💡 Simple Screen Editor とは？

Simple Screen Editor は、80x25 固定テキストセル環境向けの GUI スクリーンエディタです。 韓国語/英語が混在する入力でも文字幅の揃えを安定して維持します。

編集コアは Go で実装され、デスクトップシェルには Wails を使用しています。 単なる試験的ツールではなく、再現可能なテキストセル編集基盤の構築を目指しています。

✨ 主な機能

📐 80x25 固定グリッド

一貫したセル単位編集のため、画面を 80 列 x 25 行に固定します。

🇰🇷🇺🇸 韓国語/英語の幅揃え

文字幅を認識し、挿入/削除時もカーソル位置とセル配置を安定して維持します。

📋 クリップボード連携

OS クリップボードのコピー/貼り付けに対応し、実用的な編集フローでそのまま使えます。

💾 UTF-8 ファイル編集

UTF-8 テキストの読み込み/保存に対応し、クロスプラットフォームでテストできます。

🖼️ スクリーンショット

🚀 クイックスタート

とても簡単です。実行ファイルは 1 つだけで十分です。 ターミナルで以下のように実行するか、ファイルをダブルクリックしてください。 

simpleeditor

複雑な設定は不要です。単一の実行ファイルだけで使い始められます。
